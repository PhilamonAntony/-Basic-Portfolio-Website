Personal Portfolio Website

A simple personal portfolio website built using HTML and CSS. This project serves as an introduction to my profile, skills, and interests while demonstrating fundamental web development concepts.

Features
Home (Index) Page – Provides an overview of my profile and portfolio.
Contact Page – Allows visitors to find my contact information and connect with me.
Clean and simple user interface.
Easy navigation between pages.
Built entirely with HTML and basic CSS.
Technologies Used
HTML5
CSS3
Purpose

The goal of this project is to practice and showcase basic front-end web development skills, including page structure, styling, navigation, and layout design. It is a beginner-friendly project that demonstrates the fundamentals of creating a multi-page website.
